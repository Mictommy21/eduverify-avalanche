# EduVerify

EduVerify is a blockchain-powered certificate verification platform built on Avalanche Fuji Testnet. The platform enables educational institutions to issue tamper-proof digital certificates while allowing employers and third parties to verify their authenticity instantly.

## Problem

Academic certificate fraud is a growing challenge. Traditional verification processes are slow, manual, and vulnerable to forgery.

## Solution

EduVerify stores certificate records on the Avalanche blockchain, making them immutable, transparent, and easy to verify.

## Features

* Connect Core Wallet
* Issue student certificates
* Verify certificates using Certificate ID
* Store certificate records on Avalanche Fuji Testnet
* View blockchain transaction on Snowtrace
* Responsive modern web interface
